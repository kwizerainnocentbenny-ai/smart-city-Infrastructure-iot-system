# 🌆 Smart City IoT System

A distributed IoT system designed for monitoring environmental conditions and controlling infrastructure such as street lighting across multiple nodes.

---

## 🚀 Features

* Multi-node IoT architecture
* Real-time monitoring of environmental data
* Remote control of city infrastructure
* Scalable and modular system design

---

## 🧠 Architecture

Multiple ESP32 Nodes → Backend → Database → Dashboard

---

## 🛠️ Technologies

* ESP32
* FastAPI
* HTML, JavaScript
* MySQL / MariaDB
* HTTP / MQTT

---

## ▶️ How to Run

```id="run3"
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```
