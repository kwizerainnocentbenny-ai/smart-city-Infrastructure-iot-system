# 🌆 Smart City System Design

## Overview

This system simulates a distributed IoT infrastructure for monitoring environmental conditions and controlling city systems such as street lighting.

---

## Architecture

Multiple IoT Nodes → Backend → Database → Dashboard

---

## Key Features

* Multi-device communication
* Centralized monitoring
* Remote control of infrastructure
* Scalable architecture

---

## Data Flow

1. Each node collects environmental data
2. Data is sent to backend
3. Backend aggregates and stores data
4. Dashboard displays real-time information

---

## Scalability

* Supports multiple nodes across different locations
* Can be extended to real smart city deployments
* Integrates additional services (waste, traffic, security)
